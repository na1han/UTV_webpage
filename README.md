# EV Property UTV Webpage

Static single-page marketing site for the EV Property UTV. Built with vanilla HTML, CSS, and JavaScript for quick setup.

## Getting started
1. Open `index.html` in a browser, or run a lightweight server:
   ```bash
   python -m http.server --directory . 8000
   ```
2. Visit `http://localhost:8000` to view the page.

## Structure
- `index.html` – page layout and content.
- `styles.css` – typography, palette, and layout styling.
- `script.js` – smooth navigation and animated power readout.

## Design cues
Colors, tone, and copy are derived from the EV Property UTV marketing brief in `Webpage brief.rtf`.
